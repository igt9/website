FairWater DMA Planning

Urban water distribution modeling focused on District Metered Area (DMA) planning and Non-Revenue Water (NRW) analysis to support monitoring, pressure management, and leakage reduction.

GitHub Jump to Overview

Overview

This project supports urban water distribution analysis by developing a DMA planning workflow and applying NRW analysis concepts. The aim is to partition the network into manageable zones for monitoring and control, improve operational visibility, and identify leakage-prone behaviors through pressure/flow interpretation.

System / Study Context

The work is designed for a typical urban distribution network where supply nodes, demand nodes, and pipelines are represented in a hydraulic model. DMA boundaries are defined to create controllable zones and support targeted monitoring via meter locations.

Methodology

The workflow combines network data preparation, hydraulic simulation, DMA boundary definition, monitoring design, and NRW-focused interpretation.

Key Components

  • Network Data: Nodes, pipes, reservoirs/tanks, demands (and patterns if available)
  • Hydraulic Modeling: Flow and pressure simulation under representative conditions
  • DMA Design: Boundary selection, zone creation, inlet/outlet identification
  • Monitoring: Candidate meter locations for district-level control
  • NRW Analysis: Pressure/flow behavior and indicators supporting leakage investigation
